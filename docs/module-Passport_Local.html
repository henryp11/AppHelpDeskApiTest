Creo nuevo constructor para la estrategia deseada Requiere como parametros un objeto con los datos a validar, en este caso el nombre de usuario determinado por el correo y la contraseña Y como segundo parámetro una función asíncrona, la cual requerirá un email y password así como una función de validación "done" propia de Passport, que indica cuando todo es OK
Example
async (email, password, done) => {
    try {
      const user = await service.getUser(email, password);
      done(null, user);
    } catch (error) {
      done(error, false);
    }
